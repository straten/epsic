# epsic
Electromagnetic Polarization Simulation in C++

To compile the code,

```
./bootstrap
./configure
make
```

The project and a PDF user manual is hosted at [https://straten.github.io/epsic/](https://straten.github.io/epsic/) 
and the C++ API is documented at [https://epsic.readthedocs.io/en/latest](https://epsic.readthedocs.io/en/latest)
